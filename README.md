
# WeatherBaba

WeatherBaba is a React app which fetch and shows weather using www.weatherapi.com api. You can search for any place and see its weather, condition, temperature, uv index, etc. Simply search for weather get it in a jiffy. Its fast and realiable.


## Features

- Fast weather search
- Alert messages
- Searchbox




## Run Locally

Clone the project

```bash
git clone https://github.com/aryan-nigamoct2/WeatherBaba.git
```

Go to the project directory

```bash
cd my-project
```

Install dependencies

```bash
npm install
```

Start the server

```bash
npm run start
```
